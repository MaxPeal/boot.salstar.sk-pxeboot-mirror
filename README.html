SAL's PXE BOOT

Can boot Linux, BSD, DOS and tools like hdt, memtest, ...

Supported operating systems:
  • RPM based Linux distributions:
    • Fedora *
    • CentOS *
    • Scientific linux *
    • openSUSE
    • Mandriva
  • DEB based Linux distributions:
    • Debian
    • Ubuntu
  • Other Linux distributions:
    • Slackware
    • ArchLinux
  • Live distributions:
    • Parted Magic *
    • Clonezilla *
  • BSD systems:
    • OpenBSD
    • NetBSD
    • FreeBSD
    • PC-BSD
  • FreeDOS images:
    • UBCD FreeDOS *
    • Ultimate boot CD *
    • FreeDOS 1.1 src (installation iso) *
  • Tools:
    • Hardware detection tool (HDT) *
    • MemTest86+ *
Systems with asterisk (*) are signed and signature is verified before boot.

Requirements: *86* PC with network connection.

Optional but recommended: network configuration from DHCP server.

See into ipxe/ dir for how to boot your favorite system.
